# 🛒 Instacart - Grocery Delivery App

A full-stack grocery delivery platform built with Next.js, MongoDB, Cloudinary, Inngest, and Stripe.

## 🚀 Features

- User Authentication
- Browse Products by Category
- Search Products
- Shopping Cart
- Order Management
- Admin Dashboard
- Product CRUD Operations
- Cloudinary Image Uploads
- Responsive Design

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/grocery-delivery-app.git

cd grocery-delivery-app

npm install

npm run dev
```

---

## 🔑 Environment Variables

Create a `.env.local` file:

```env
MONGODB_URI=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
INNGEST_EVENT_KEY=
INNGEST_SIGNING_KEY=
STRIPE_SECRET_KEY=
```
